# Global IP Checker

## Requirement

- OS: Mac, Windows
- python3
  - datetime
  - requests
  - platform
  - os (Mac)
  - plyer (Windows)



## Tested Environment

- Mac
  - macOS Big Sur 11.5.2 on M1 MacBook Air 2020
- Windows
  - Windows 10 pro (Insider program, 21354.1) on Parallels on M1 MacBook Air 2020
  - Windows 10 pro on Desktop PC
